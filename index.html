Edit It!

Simple jQuery plugin for fast inline edit.

Usage

JavaScript

options = {"url": "/change-name"};
$(".double-click").editit("dblclick", options);
$(".click").editit("click", options);

HTML

<table>
  <tr>
    <th>Column 1</th>
    <th>Column 2</th>
  </tr>
  <tr>
    <td class="double-click" data-name="value-1">Double Click Me!</td>
    <td class="click" data-name="value-2">Just Click Me!</td>
  </tr>
</table>

Result:

Column 1 Column 2
Double Click Me! Just Click Me!
